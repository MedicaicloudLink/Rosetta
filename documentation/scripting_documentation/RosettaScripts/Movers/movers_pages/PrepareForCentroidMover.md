## PrepareForCentroid
*Back to [[Mover|Movers-RosettaScripts]] page.*

Convert a pose into centroid mode in preparation for low-resolution loop 
modeling.  This is used internally by LoopModeler.

```xml
<PrepareForCentroid name="(&string)"/>
```
