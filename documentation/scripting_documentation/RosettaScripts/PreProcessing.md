#Preprocessing in Rosetta Scripts

Every rosetta script is run through a preprocessor that
allows you to have simple commands in your scripts.
These are similar to [[script_vars|RosettaScripts.md#options-available-in-the-xml-protocol-file_variable-substitution]]
but allow for more control.
